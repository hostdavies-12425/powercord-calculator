# powercord-calculator
A plugin that you can use to calculate in discord

> `requirements:`
- Powercord
- Node.js
- Git

> `installation:`
```yml
cd powercord\src\Powercord\plugins && git clone https://github.com/hostdavies-12425/powercord-calculator && cd powercord-calculator && npm i
```
